# Frontend Developer Assessment Task

This project is a responsive frontend UI implementation based on the provided Figma design as part of a Frontend Developer assessment task. The main goal of this project is to convert the given design into a clean, responsive, and user-friendly interface while maintaining proper code structure, best practices, and performance optimization.
